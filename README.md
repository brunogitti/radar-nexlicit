# Radar NexLicit

Captura editais com proposta em aberto no Portal Nacional de Contratações Públicas (PNCP), em uma lista fixa de municípios de SP, MG e MS, filtra por palavra-chave e etiqueta cada edital com o segmento de produto correspondente.

Projeto em construção, camada por camada. Detalhes de arquitetura e decisões em `docs/` (a criar conforme o projeto avança).
